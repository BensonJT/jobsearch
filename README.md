# Jobsearch

A Python-based application to automate job searches and track applications.

## Project Structure
- `backend/`: Python backend services.
- `frontend/`: Frontend application (to be determined).
- `docs/`: Project documentation.

## Getting Started
- Create and activate the conda environment: `conda create -n jobsearch python=3.12`
- Install dependencies: `pip install -r requirements.txt`
